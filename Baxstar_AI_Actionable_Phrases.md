# Baxstar AI Actionable Phrases
*Trigger phrases for AI-executed workflows across Claude, ChatGPT, and Gemini*
*Updated: April 29, 2026*

---

## How To Use
Say the phrase exactly as written. The AI will locate the corresponding instruction file in Google Drive → Baxstar Data Engine → AI Inputs → Asset Pipeline and execute the defined steps.

---

## Asset Phrases

| Phrase | What It Does |
|---|---|
| `Run asset pipeline` | Compress image (Squoosh) + compress video (Handbrake) + upload both to Wix + update hero HTML with new CDN URLs |
| `Run Step 1 only` | Image compression via Squoosh only — outputs WebP at correct size and quality |
| `Run Step 2 only` | Video compression via Handbrake only — outputs web-optimized MP4 |
| `Run Step 3 only` | Upload assets to Wix Media Manager + return CDN URLs |

---

## Site Phrases

| Phrase | What It Does |
|---|---|
| `Baxstar sync` | Sync full context — brand standards, tech stack, current state — across AI sessions |
| `Build new section` | Build a new Wix Custom Embed HTML section following brand standards, SEO standard, and frontend design skill |
| `Run SEO check` | Audit current HTML section against SEO Optimization Standard — return list of gaps |
